# Debsacker

[![Build status](https://travis-ci.org/konstantinous/debsacker.svg)](https://travis-ci.org/konstantinous/debsacker)

## Installation

Add this line to your application's Gemfile:

    gem 'debsacker'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install debsacker

## Usage

`debsacker --help`
